# Free Tools

## Definition

Free utility suite that drives tradesman signups to the paid [[Intake Engine]]. Acts as funnel top — tradesmen find tools via search, use them, see the filter link offer, convert.

## Current Free Tools

| Tool | Function |
|---|---|
| Quote calculator | Estimate job quotes from inputs |
| Job estimator | Time and cost estimation |
| Diesel calculator | Fuel cost per job |
| Lead scans | 2 scans per week (lead generation preview) |
| Newsletter | Tips and trade news |

## Funnel Logic

1. Tradesman searches "quote calculator UK" or similar
2. Lands on free tool page
3. Uses tool (gets value)
4. Sees soft CTA: "Stop chasing ghosts — get your filter link"
5. Click → [[Intake Engine]] signup

## Why This Layout

- Free tools = SEO acquisition without ad spend
- Tools demonstrate JobFilter understands trade workflow
- Building trust before asking for £49/month
- Each tool = potential landing page for paid Google traffic later

## Rules

- Every free tool must end with subtle [[Intake Engine]] push
- Free tier never replaces core product — only supports funnel
- Limits (2 scans/week) create natural upgrade pressure
