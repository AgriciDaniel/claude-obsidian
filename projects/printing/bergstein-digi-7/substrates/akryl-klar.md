# Klar akryl (PMMA)

Förlåtande substrat. Akryl tar UV-bläck väldigt bra och primer behövs sällan vid ren yta.

## Settings

| Parameter | Värde |
|---|---|
| Primer | 30% (testa 0% först) |
| White combined | 230% |
| Per white-kanal | 77% |
| CMYK | 80% |
| TAC per strike | 300% |
| Inter-layer dwell | 5s |
| Adhesion test mål | 5B |

## Yt-prep

1. IPA wipe — lätt, akryl tål inte starka lösningsmedel
2. **INTE använd aceton** — löser akryl och skapar microcracks
3. Avlägsna skyddsfilm precis innan tryck (inte tidigare — då samlar ytan damm)
4. Hantera med handskar

## Anteckningar

Akryl är värmekänsligt — hög UV-cure-intensitet kan deformera tunna plattor. Kör ECO-mode cure om plattan är <3mm.

Akryl är repkänsligt — packa printprodukter med mellanlägg eller skyddsfolie igen efter print.

Akryl reagerar med vissa lösningsmedel i UV-bläck (sällsynt med moderna formuleringar, men test-wedge på ny ink-batch ALLTID). Symtom: dimma eller grumlighet runt printen.
