# Bar aluminium (otreated)

Krävande substrat. Slät, icke-porös, oxidationsfilm — primer och mekanisk prep är obligatoriskt.

## Settings

| Parameter | Värde |
|---|---|
| Primer | 70% |
| White combined | 220% |
| Per white-kanal | 73% |
| CMYK | 80% |
| TAC per strike | 300% |
| Inter-layer dwell | 8s (längre — primer behöver tid på metall) |
| Adhesion test mål | 4B minimum, 5B önskat |

## Yt-prep (kritisk)

1. IPA wipe, lint-free, 60s
2. **Scuffa lätt** med #0000 stålull eller 600-grit slippapper för mekanisk key
3. IPA igen efter scuff (avlägsnar slipdamm)
4. Hantera med handskar

## Anteckningar

Adhesions-failure mode = mest sannolikt här av alla metaller utom stainless. Test wedge + cross-hatch på varje ny batch ALDRIG hoppa över.

UV-cure intensitet bör vara +10–15% jämfört med dopplackad metall — metallen leder bort värme från cure-zonen.
