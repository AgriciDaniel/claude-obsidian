# Dopplackad metall

Standardsubstrat för produktion. Dippad/lackerad metall — beter sig som plast snarare än som ren metall vid UV-tryck. Lackskiktet är där bläcket fastnar; metallen under spelar mindre roll.

## Settings (verifierat)

| Parameter | Värde |
|---|---|
| Primer | 30% (testa 0% först) |
| White combined (3 heads) | 180–200% |
| Per white-kanal | 65% |
| CMYK | 80% per kanal |
| TAC per strike | 300% |
| Inter-layer dwell | 5s |
| Adhesion test mål | 5B (cross-hatch) |

Om lacker är ljus (vit, beige): sänk white combined till 150%.

## Yt-prep

1. IPA (≥99%) wipe, lint-free, 60s flash-off
2. **INTE** scuffa — skadar lacket och bryter korrosions-sealen
3. Om delen är >6 månader gammal: dubbel-IPA, 5+ min vänta
4. Hantera med bomullshandskar efter prep

## Vanligt fel

Silikonkontamination → fish-eyes. Cure med mild detergent + IPA-double-clean.

## Anteckningar

Närmast plast-beteende av alla metallsubstrat. Snäll. Bra startsubstrat för en ny operatör.
