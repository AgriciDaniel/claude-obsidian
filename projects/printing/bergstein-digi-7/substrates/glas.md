# Glas

Helt transparent → maximal vit-täckning krävs. UV-bläck fastnar bra på rent glas med rätt primer.

## Settings

| Parameter | Värde |
|---|---|
| Primer | 30% |
| White combined | **250%** (max för opacitet) |
| Per white-kanal | 83% |
| CMYK | 80% |
| TAC per strike | 280% |
| Inter-layer dwell | 5s |
| Adhesion test mål | 4B–5B |

White combined-värdet är högsta av alla substrat. Glas blockerar inget ljus själv så vit-blockering är allt.

## Yt-prep

1. IPA wipe, dubbel-runda för glas (mer fingerprints och rester än metall typiskt)
2. Använd lint-free torkare specifikt avsedda för glas (microfiber, INTE pappersservetter — fibrer hänger kvar)
3. Hantera med kanthandskar eller sugkoppar — undvik kontakt med tryckytan
4. Snabbtest: andas på glaset → om imman skingras direkt och inte droppar → ren yta

## Anteckningar

Glas-print som faller av efter installation: oftast pga kondens i printapplikationen (utomhusmiljö). UV-bläck är inte värmebeständigt över ~80°C — undvik glas i ugn / diskmaskin om du inte använder dishwasher-rated UV-formulering.

Krökt glas (drinkware) kräver extra noggrann head height-kalibrering — för långt avstånd → mistsprid → suddiga kanter.
